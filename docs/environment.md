# 환경 변수

이 문서는 현재 코드 기준으로 자주 사용하는 환경 변수를 정리한 것입니다. 민감한 값은 가능하면 직접 값 대신 `*_FILE` 방식을 권장합니다.

## 필수

- `DB_PASSWORD` 또는 `DB_PASSWORD_FILE`
- `JWT_SECRET` 또는 `JWT_SECRET_FILE`

`DB_PASSWORD`는 비어 있으면 서버가 기동하지 않습니다. `JWT_SECRET`은 최소 32자 이상이어야 합니다.

## 애플리케이션 기본값

- `APP_ENV`
- `CODE_PLATFORM_DATA_DIR`
- `CODE_PLATFORM_USERS_DIR`

## 운영 권장

- 민감한 값은 직접 `.env`에 쓰기보다 `*_FILE`로 주입합니다.
- 도메인 HTTPS를 쓸 때는 `GOOGLE_OAUTH_ALLOWED_REDIRECT_URIS`에 실제 도메인 callback URL을 포함합니다.
- 운영에서는 `ANALYSIS_QUEUE_MODE=rq`, `PROBLEM_FOLLOW_UP_QUEUE_MODE=rq`를 권장합니다.
- 관리자 shutdown 기능은 정말 필요한 환경에서만 `CODE_PLATFORM_ENABLE_ADMIN_SHUTDOWN=true`로 엽니다.
